># 오라클 내장 함수
>### 숫자, 날짜/시간, 문자, NULL, 조건
>### 집계, 순위, 순서 
>### 분석

## 숫자
+ ``ABS(숫자)`` : 절댓값　　　　``SQRT(숫자)``: 제곱근　
+ ``CEIL(숫자)``: 올림　　　　``FLLOR(숫자)``: 내림
+ ``ROUND(숫자,n)``: 소수점 n자리 기준 반올림
+ ``POWER(숫자,n)``: n제곱
+ ``TO_CHAR(숫자, '형식문자')``: 숫자 -> 문자 ★


## 날짜/시간

+ ``SYSDATE``: 오라클 서버상의 날짜/시간

+ ``SYSDATE + 9/24``: 우리나라 날짜/시간
+ ``NEXT_DAY(날짜, '요일')``: 지정한 주의 요일 날짜

+ ``ADD_MONTH(날짜, N)`` : N달 뒤 날짜

+ ``LAST_DAY(날짜)`` : 지정한 달의 마지막 날짜

+ ``TO_CHAR(날짜, '형식문자')`` : 날짜 -> 문자 ★


## 문자

+ ``LOWER('문자')``: 소문자　      ``UPPER('문자')``: 대문자
+ ``INITCAP('문자')``: 첫 글자 대문자

- ``ASCII('문자'or숫자)``: ASCII 번호

+ ``CHR(ASCII 번호)``: ASCII 번호 문자

- ``LENGTH('문자')``: 길이　　　　　``LENGTHB('문자')``: 길이(BYTE)

+ ``SUBSTR('문자', 위치, 길이)``: 추출

- ``REPLACE('문자', '변경전문자', '변경후문자')``: 변경

+ ``INSTR('문자', '찾을문자', 찾을위치, 문자개수)``: 검색

- ``LTRIM('문자')``: 왼쪽 공백제거
- ``RTRIM('문자')``: 오른쪽 공백제거
- ``TRIM('문자')``: 양쪽 공백제거

+ ``LTRIM('문자', '제거문자')``: 왼쪽 문자제거
+ ``RTRIM('문자', '제거문자')``: 오른쪽 문자제거
+ ``TRIM('문자', '제거문자')``: 양쪽 문자제거

- ``concat('문자', '문자')``: 연결(2개)
- ``'문자' || '문자' || '문자'``: 연결(여러개)

+ ``L,RPAD('문자', 전체자리수, '추가할문자')``: 추가


- ``TO_NUMBER('문자')``: 문자 -> 숫자 ★

+ ``CAST('문자' AS NUMBER)``: 문자 -> 숫자

- ``TO_DATE('문자', '형식문자')``: 문자 -> 날짜 ★


## NULL
+ ``NVL(속성, 값)``: NULL -> 값
+ ``NVL(속성, 값1, 값2)``: NOT NULL -> 값1 , NULL -> 값2
+ ``NULLIF(속성(문자), '문자1')``: NULL -> 문자1
+ ``COALESCE(속성(숫자), 숫자)``: NULL -> 숫자

## 조건
```
DECODE(속성, 조건값1, 결과1, 조건값2, 결과2, ... ,결과n)

CASE WHEN 조건1 THEN 결과1
     WHEN 조건2 THEN 결과2
          ...
     ELSE 나머지조건
     END new_테이블_이름
```
